Summary
The Excel COUPDAYS function returns the number of days in a coupon period that includes the settlement date.
Purpose 
Get days in coupon period incl settlement date
Return value 
Calculated days
Arguments 
settlement - Settlement date of the security.maturity - Maturity date of the security.frequency - Number of coupon payments per year (annual = 1, semi-annual = 2, quarterly = 4).basis - [optional] Day count basis (see below, default =0).
Syntax 
=COUPDAYS(settlement, maturity, frequency, [basis])
